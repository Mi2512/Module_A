## Обзор проекта

Реализпция агента для сбора и предварительной обработки данных о маршрутах с обработкой геодезических поправок и анализа окружающей среды.

## Модули

- **Модуль A.1: Загрузка данных с обработкой геодезических поправок**
  - Загрузка треков с обработкой геодезических поправок
  - Обнаружение и исправление аномалий в координатах
  - Хранение в базе данных с версионностью

- **Модуль A.2: Формирование набора данных с универсальной расшифровкой карт**
  - Извлечение данных об окружающей среде в радиусе 500 метров
  - Универсальный алгоритм расшифровки обозначений
  - Классификация типов местности и объектов

- **Модуль A.3: Предобработка данных и выделение значимых атрибутов**
  - Обнаружение и обработка аномалий
  - Выделение значимых атрибутов с помощью анализа корреляций
  - Обеспечение всесезонности данных

- **Модуль A.4: Описание структуры набора данных**
  - Описание атрибутов с единицами измерения
  - Проверка нормальности распределения
  - Анализ типов распределений

- **Модуль A.5: Расширение набора данных**
  - Дополнение треками из туристических платформ
  - Аугментация изображений с сохранением географической достоверности

- **Модуль A.6: Подготовка отчетов**
  - Отчёт автоматом генерится
